**[PROGRAM DESCRIPTION]**

	MyRecipe is an application for managing and searching for cooking ideas based
	on available ingredients. The primary data used is recipe data, ingredients, and
	cooking categories. The application's users are the public and cooking
	enthusiasts.

**[SPECIFICATION]**

  	A. Users can add, edit, and delete recipe data.
	
  	B. The system can record ingredient composition, preparation steps, and
  	estimated cooking time.
	
  	C. Users can search for recipe data by main ingredients using Sequential and
  	Binary Search.
	
  	D. Users can sort recipe data by cooking time or alphabetically by title using
  	Selection and Insertion Sort.
	
  	E. The system can display statistics on the number of recipes per ingredient
  	category and a list of the most frequently searched menus
